# HTML/CSS Projects (Showcase)

Три учебных проекта по вёрстке (HTML/CSS).  
Эти проекты **не ключевые для Data Analyst**, но полезны как подтверждение: аккуратность, внимание к деталям, умение доводить до результата.

## Проекты
1) **Оно тебе надо** — лендинг аукциона  
2) **Надо сделать идеально** — интерактивный лендинг (tooltip)  
3) **Посмотри в окно** — страница с видео, формой поиска и шаблонами карточек (как в оригинальном задании)

## Как посмотреть в браузере
- Скачай репозиторий и открой любой `index.html` в папках проектов.
- Или включи GitHub Pages: **Settings → Pages → Deploy from branch → main /root**  
    После этого будет доступно: https://JuliaMamaMia.github.io/html-css-projects/
